CARGAR EL CONTENIDO DE UN DOCUMENTO A UNA VARIABLE EN BASH

En ocasiones se desea revisar, modificar, procesar o sumar el contenido de un documento a otras variables. Para esto, las siguientes líneas facilitan el trabajo:

vrbl=$(cat documento.txt)
echo $vrbl

El comando cat extrae el contenido de documento.txt y luego dicho contenido es guardado en la variable vrbl.

~ finis ~
