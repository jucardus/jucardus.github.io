## Caracteres no admitidos o reservados

Caracteres reservados


```
< (menor que)
> (mayor que)
: (dos puntos)
" (comilla doble)
/ (barra diagonal)
\ (barra diagonal inversa)
| (barra vertical o canalización)
? (signo de interrogación)
* (asterisco)
```

Caracteres permitidos

```
!#$%&'()+,-.;=@[]_{}~
```
