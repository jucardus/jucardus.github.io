History of Swap Space & Virtual Memory

From early computing to modern Linux swap files

The Concept of Virtual Memory

Virtual memory is a memory management technique that provides an "idealized abstraction of the storage resources that are actually available on a given machine." The concept was developed in the late 1950s and early 1960s at a time when physical memory was extremely expensive and limited.

The key idea behind virtual memory is that it allows programs to execute without being completely loaded into physical memory, with portions of programs stored on disk and loaded as needed.

Early Pioneers

The first documented implementation of virtual memory was in the Atlas computer, a joint development between the University of Manchester and Ferranti Ltd. in the United Kingdom. The Atlas is considered the first computer with virtual memory, implemented in 1959.

Tom Kilburn

Lead architect of the Atlas computer, which implemented the first virtual memory system called "one-level storage system."
